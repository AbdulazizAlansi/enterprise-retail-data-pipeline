# Enterprise Retail Data Pipeline

## Project Overview
An end-to-end enterprise Data Engineering project that simulates a real-world retail analytics pipeline using modern data technologies and ETL orchestration tools.

## Technologies Used
- Apache Kafka
- Apache Airflow
- Talend Open Studio
- PostgreSQL
- PySpark
- Great Expectations
- Power BI
- Docker
- Python

## Architecture
This project follows a modern enterprise data pipeline architecture:

Raw Dataset → Kafka → Airflow → PostgreSQL Staging → Talend ETL → PySpark Transformations → Data Warehouse → Power BI

## Project Goals
- Build scalable ETL pipelines
- Implement data quality validation
- Design a star schema warehouse
- Simulate enterprise batch & streaming workflows
- Generate business intelligence dashboards
